Description
[List all Users or list all Dir&Files site by site for Alfresco CE >5.0]
Uniquement pour Alfresco CE version 5 minimum. Permet de lister les utilisateurs site/site avec plusieurs détails. Ou présente les fichiers et les répertoires pour un site au complet, avec ou sans options d'affichage.

A installer si possible là ou Alfreco est installé
Dépendances : PHP > 5 + PHP-CURL

Utilise les requêtes CMIS via PHP-CURL pour interroger Alfresco, les reste est une mise en forme pour l'affichage.

Le répertoire d'installation peut-être changé, ces pages fonctionnent bien chez moi, néanmoins j'ai un serveur Apache en plus d'Alfresco, pour diffuser ces pages.

La version 5.2 d'Alfresco ne prend plus en chargele listing des fichiers et des répertoires...à suivre.
